Lab 4 - GIS4035 - Spatial Enhancement and Band Indicies

For this week’s lab, after going through several exercises about using the different spatial filters, using histograms, and using band indicies in ERDAS imagine (and to a lesser extent in ArcGIS,) we created 3 output maps based on three mystery features described in this week’s lab. I identified Mystery Feature #1 as “water,” in particular the inlets in the upper-righthand corner of our image to be analyzed, Mystery Feature #2 as snow/ice in the mountains, and Feature #3 as the bay which showed high brightness in Layers 1-3, and to a lesser extent 4, but no brightness variation from the rest of the water in Layers 5-6.
