GIS 4006 - Module 2 Lab - Typography

By Kristin

March 25, 2020

This week’s assignment was to create a map of Marathon, Florida and label it “by-hand” in Adobe Illustrator, using the ArcGIS plugin to load the map (from a simple basemap imported from the FGDL in ArcGIS and then exported to a web map for ArcGIS Online.) Previously, I had only used ArcGIS for labeling maps automatically. The lecture and reading covered the standards for labeling map features, including standards for points, lines, and areas, and mentioned to make sure to change the labels accordingly to make ordinal or nominal differences obvious (e.g. make the text for a larger city larger and make the symbols for a city and an airport different.) In an odd way, this was almost mind-blowing for me. I’d never even noticed that all maps have italicized text for water features, and that different types of features tend to have different fonts or styles or colors or cases to distinguish them from each other.

The key tools I used in Adobe Illustrator after syncing the map board to the document were the text tools, line tools, symbol tools. I spent a lot of time modifying the text size, color, case, and rotation to match the shape of the land.

I need to find a better way to do scale bars in the future, but for this assignment, I picked a reasonable maximum size in miles for the scale bar, then converted that to pt (the units Adobe Illustrator automatically displayed the length of the scale bar in) via inches, and then resized the scale bar to that pt size. Then, I labeled it by-hand with the text tool. Similarly, I created the legend by placing symbols and labels next to each-other by-hand, and my alignment is slightly off.


Map of Marathon, Florida created by me, Adobe Illustrator, and the ArcGIS extension for Adobe Illustrator.

I made three customizations to this map: a drop-shadow for the main series of islands to draw attention to it and create an appealing 3D-effect, rounded corners for the inset map and the legend to give a more smooth look, and used different symbols for the airport, country club, state park, large city, and smaller cities to, again, visually draw an obvious distinction between them.
