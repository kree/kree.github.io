GIS 4048 - Lab2 - Forestry and Lidar

In this lab, LIDAR data was used to generate a map of the tree canopy density for a region of Shenandoah National Park, Virginia. This area lies in the Blue Ridge Mountains and along the Appalachian Trail. Here’s the National Park Service map for part of this area: https://www.nps.gov/shen/planyourvisit/upload/BigMeadows_RoadTrail.pdf ! Lidar data was provide in Lidar Point Cloud (LPC) format. This was used to create a DEM by filtering to include only ground points, and a DSM representing the canopy, which included only non-ground points.

GIS 4048/5100 - Lab 1 Least Cost Path Analysis and Corridor Analysis: Scenario 2

For this section of Lab 1, a suitability assessment for property development was performed in an area of Southern Oregon. First, information provided about soil and land types via raster images were reclassified into rasters with cell values indicating suitability ratings for development. Next, suitability ratings were assigned to slope values, using information from a DEM. Following this, rasters were generated indicating the distance from rivers and roads in this study area and these were also reclassified to suitability ratings.

GIS 4048 - Lab 1 - Scenario 4: Coronado National Forest Black Bear Corridor Analysis

For this section of the first Lab, “Scenario 4,” the most suitable or least-cost corridor for black bear movement was calculated between two protected areas of Coronado National Forest in Southern Arizona. Criteria included in the suitability calculation were: elevation, distance from roads, and land cover types. ArcGIS Pro was used to create Cost Distance surfaces from each Protected Area, and these were then combined using the Corridor tool to generate a single corridor.

GIS 4006 - Module 6 - Isarithmic Mapping

This week’s topic was Isarithmic mapping – that is, mapping continuous values like precipitation or elevation. Israrithmic maps can feature continuous tones (unclassed) or hyposometric tints (classed, and usually separated by contour lines.) This week’s goal was to map average annual precipitation from 1981 - 2010 in Wasington State, USA. We created maps using both continuous and hypsometric tinting and added a topography-based hillshade to add texture and interest to the maps.
