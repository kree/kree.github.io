GIS 5935 - Lab 5 / Module 2.2: Surface Interpolation

In this lab, we examined different methods for interpolating between points, including: Inverse Distance Weighting (IDW,) Thiessen (Nearest Neighbor), and Spline. Within Spline we investigated both the Tension and Regular options offered by ArcGIS. In Part A, we examined a set of elevation points in Kansas and created interpolated DEMs from them using IDW and Spline (tension) interpolations. Then, we subtracted the results using the Raster Calculator to analyze the differences between the results.

GIS 5935 - Lab 4 / Module 2.1: Surfaces - TINs and DEMs

In this week’s lab, we examined digital elevation models (DEMs) and surfaces represented by TINs (triangular irregular networks.) Unlike a DEM, a TIN is a geometric three dimensional model of a surface, rather than the traditional DEM raster model with a raster value for elevation at each pixel. The TIN model allows for geometric operations like intersection with a triangular facet to determine what is visible from a point, that would be more difficult to compute from a raster DEM surface.

GIS5935 - Lab 1: Calculating Metrics for Spatial Data Quality

For the first lab of GIS5935, we examined accuracy and precision via two datasets. In the first, 50 measurements were taken of a single point using a Garmin GPS unit. First, the mean waypoint was calculated and placed on the map using the means of the x and y values of all the waypoints. Then, the distance of each waypoint from the average waypoint was calculated. Horizontal accuracy was calculated as the distance (measured using a tool in ArcGIS) between the mean waypoint measurement and a provided reference waypoint.
